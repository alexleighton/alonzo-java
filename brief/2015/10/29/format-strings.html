Format-String All the Things! Brief

October 29, 2015 • Alex Leighton

This post is based on code found in commit 1f0b04e7192

I’ve picked up a practice recently: any time I need to provide a message (typically for errors or logging), I make sure the function in question takes varargs and then pass them to String.format. Most of the time I provide a message, I eventually want to use a format-string and it’s just lame to have to call String.format every time.
